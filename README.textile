To use in Shopify to add a shipping calculator to your cart page.

h2. Dependencies

jQuery > 1.3
jQuery template plugin ("http://api.jquery.com/category/plugins/templates":http://api.jquery.com/category/plugins/templates)
shopify_common.js
countries.js
api.jquery.js

h2. Installation

"http://wiki.shopify.com/Adding_a_Shipping_Rates_Calculator_to_your_Cart_Page":http://wiki.shopify.com/Adding_a_Shipping_Rates_Calculator_to_your_Cart_Page

h2. Configuration

Configuration parameters are all optional.

You pass an object to Shopify.Cart.ShippingCalculator().
