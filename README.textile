To use in Shopify to add a shipping rates calculator to the cart page. Relies on a JSON API to fetch rates based on country, state/province and zip code.

h2. Dependencies

jQuery > 1.3
jQuery template plugin ("http://api.jquery.com/category/plugins/templates":http://api.jquery.com/category/plugins/templates)
shopify_common.js
countries.js
api.jquery.js
